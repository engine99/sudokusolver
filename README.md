Sudoku Solver - an Angular application

An Angulart
Implements a depth-first search algorithm.

Set up environment
> npm install -g angular
> npm install -g @angular/cli
> npm install -g cordova
> npm install -g angular-ide

To build:
> cd sudoku-solver/src
> ngide open .
> cd ..
> ng build
> cd ..
> cordova build


To deploy:
> cordova run
